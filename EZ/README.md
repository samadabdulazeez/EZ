# EZ Job Portal

A Streamlit-based dashboard for job and grant listings, designed to help users discover opportunities.

## Features

- Recent Grant and Job listings
- Featured opportunities with categorized sections
- Navigation arrows for browsing through listings
- Direct application links to Google Forms
- AI Job/Grant Matching section (Coming Soon)

## Installation

1. Clone this repository
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the Streamlit dashboard:
```bash
streamlit run "EZ Dashboard.py"
```

The dashboard will open in your browser at `http://localhost:8501`

## Requirements

- Python 3.7+
- Streamlit
